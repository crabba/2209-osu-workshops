=====================
Event Engine Accounts
=====================

.. _logging_in:

Event Engine provides the free, short-lived AWS accounts we will be using during this workshop.

----------
Logging In
----------

* Go to https://dashboard.eventengine.run and enter the event hash supplied at the workshop. 
* On the **Sign In With** page, select **Email One-Time Password (OTP)**, and enter the password you receive by email.
* On the **Team Dashboard** page, select **AWS Console**
* In the **AWS Console Login** popup, you can select **Open AWS Console** to proceed to the console. 
   * You can retrieve the AWS Console link later by visiting https://dashboard.eventengine.run/dashboard from the same browser you used to log in.
